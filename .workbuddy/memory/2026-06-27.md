# CatClawMusicServer 工作日志 - 2026-06-27

## 今日完成：C# ASP.NET Core 完全重写

### 项目结构（新建）
```
CatClawMusicServer/
├── CatClawMusicServer.csproj   ← 项目文件（.NET 8 + EF Core + TagLibSharp）
├── Program.cs                  ← 主程序（服务配置、中间件管道）
├── appsettings.json            ← 配置文件（端口、音乐目录、限速）
├── Properties/launchSettings.json
├── Data/ApplicationDbContext.cs  ← EF Core SQLite 上下文
├── Models/
│   ├── Song.cs
│   ├── Artist.cs
│   ├── Album.cs
│   ├── Playlist.cs
│   ├── PlaylistSong.cs
│   ├── Favorite.cs
│   └── PlayHistory.cs
├── Controllers/
│   ├── SongsController.cs      ← 歌曲 CRUD + 流媒体 + 封面 + 歌词
│   ├── ArtistsController.cs    ← 艺术家列表/详情/歌曲
│   ├── AlbumsController.cs     ← 专辑列表/详情/歌曲
│   ├── SearchController.cs     ← 全文搜索（分页）
│   ├── PlaylistsController.cs  ← 播放列表 CRUD + 歌曲管理
│   ├── FavoritesController.cs  ← 收藏 API
│   └── HistoryController.cs    ← 播放历史 API
├── Services/
│   ├── FileTagService.cs      ← TagLibSharp 封装（读取 14 种格式标签）
│   └── MusicScanner.cs        ← 目录扫描服务（自动入库）
└── wwwroot/index.html          ← Web UI（歌曲浏览 + 播放器 + 歌词面板）
```

### NuGet 包
- `Microsoft.EntityFrameworkCore.Sqlite` 8.0.11
- `TagLibSharp` 2.3.0（音频标签读取）
- `Swashbuckle.AspNetCore` 6.6.2（Swagger）
- `AspNetCoreRateLimit` 5.0.0（限速）

### API 端点（全部支持分页）
| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /api/songs | 歌曲列表 `?page=&page_size=&artist=&album=` |
| GET | /api/songs/{id} | 歌曲详情 |
| GET | /api/songs/{id}/stream | 流媒体（Range 支持） |
| GET | /api/songs/{id}/cover | 封面图 |
| GET | /api/songs/{id}/lyrics | 歌词（.lrc 文本） |
| GET | /api/artists | 艺术家列表（含 song_count） |
| GET | /api/albums | 专辑列表（含 song_count） |
| GET | /api/search?q= | 搜索（标题/艺术家/专辑） |
| GET/POST/PUT/DELETE | /api/playlists | 播放列表管理 |
| GET/POST/DELETE | /api/favorites | 收藏 |
| GET/POST/DELETE | /api/history | 播放历史 |

### Web UI 功能
- 左侧导航：歌曲 / 艺术家 / 专辑 / 播放列表 / 设置
- 歌曲列表：分页、当前播放高亮
- 艺术家/专辑：网格卡片 + 歌曲数量
- 搜索栏：实时搜索（300ms 防抖）
- 底部播放器：播放/暂停、进度条、音量、封面显示
- 歌词面板：浮动面板，自动加载 .lrc 文件

## 下一步

4. 按 F5 运行
5. 访问 `http://localhost:5000` 测试
